# 💰 Budget Tracker

A simple, privacy-first web application to track income and expenses in real-time — no login, no ads, no server.

### 🔴 [**▶ Try Live Demo**](https://revathiksg6.github.io/Budget-Tracker)

## Features
- Add income and expense transactions with category, date & description
- Real-time balance calculation
- Edit and delete transactions
- Pie chart visualization (income vs expense)
- Filter transactions by month
- **Monthly Budget Limit** with alert when limit is crossed
- **Savings Goal Tracker** with progress bar
- **Recurring Transactions** (auto-added every month)
- **Category Icons** (Food, Travel, Shopping, Bills, Salary, Other)
- **Dark / Light Theme Toggle**
- **Voice Input** to add transactions by speaking
- **Export as CSV (Excel)** and **Export as PDF**
- Data stored locally in browser (LocalStorage) — fully private, no server involved

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js (data visualization)
- Web Speech API (voice input)
- Browser LocalStorage (data persistence)

Open this link on any phone or laptop browser — no installation needed, works instantly.

## How to Use
1. Enter description, amount, date, and select category
2. Choose Income or Expense
3. Optionally check "Repeat every month" for recurring entries
4. Click "Add Transaction" or use "🎤 Add by Voice"
5. Set a monthly budget limit or savings goal to track progress
6. Export your data as CSV or PDF anytime

## Privacy
This app stores all data locally in your browser. No data is ever sent to a server, ensuring complete privacy without requiring login or premium access.
