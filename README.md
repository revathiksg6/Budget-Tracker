# 💰 Budget Tracker

A simple and responsive web application to track income and expenses in real-time.

## Features
- Add income and expense transactions
- Track total balance instantly
- Delete transactions
- Clean and modern UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Live Demo
[View Live App](https://revathiksg6.github.io/Budget-Tracker)

## How to Use
1. Enter a description (e.g. Salary, Food)
2. Enter the amount
3. Select Income or Expense
4. Click "Add Transaction"
5. Balance updates automatically
